## Installation

### Requiremenmts
* Python 3.3+ or Python 2.7 
* scikit learn
* numpy 
* pandas
* matplotlib

Install the libraries from the pypi.org with the following commands.
```
pip install scikit-learn
pip install numpy
pip install pandas
pip install matplotlib
```
